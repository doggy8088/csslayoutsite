關於 max-width 屬性

#main {
  max-width: 600px;
  margin: 0 auto; 
}
<div id="main">

使用 max-width 替代 width 可以更完美的處理當瀏覽器視窗小於元素寬度的情況，這點在行動裝置上顯得更為重要，你現在就可以調整瀏覽器視窗大小看看這頁的變化！

另外，max-width 屬性幾乎在所有的主流瀏覽器都有支援（包括 IE7+ 以後版本），所以各位可以不用擔心有瀏覽器相容性的問題。

</div>
